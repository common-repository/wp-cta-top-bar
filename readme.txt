== Description ==
 
Add simple and clean call to action top bar in your wordpress website. This plugin help you promote your important message in your wordpress website. 

Features:
- Set unlimited background color
- Set text color
- Cookie adjustment
- Content flexibility
- Allow shortcode in the top bar content
- Enable/Disable option
- Sticky Option
- User friendly menu
 
= Installation =
1. In your WordPress admin panel, go to Plugins > New Plugin
2. Find WP CTA Tob Bar plugin by Alipio Gabriel and click Install now
3. Alternatively, download the plugin and upload the contents of wpctatopbar.zip to your plugins directory, which usually is /wp-content/plugins/
4. Activate the plugin
